# Playbook biên tập (bản lưu trong repo)

**Nội dung đầy đủ đã được publish trên site:** `/huong-dan-bien-tap/`

**File nguồn để chỉnh sửa (khuyến nghị):** `src/pages/huong-dan-bien-tap.md`

Khi cập nhật playbook, sửa file `.md` trong `src/pages/` rồi build/deploy; file trong thư mục `docs/` này chỉ làm mốc dẫn đường cho người đọc trong GitHub.
